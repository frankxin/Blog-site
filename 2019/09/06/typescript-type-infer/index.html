类型保护

类型保护作为一种判断过滤可以让我们通过缩小范围，更精准的确认类型
常见的类型保护包括以下几种：

  • typeof
  • instanceof
  • in
  • 字面量类型保护
  • 自定义类型保护

typeof

typeof 可以用来分辨原始类型的不同

instanceof

instanceof 可以用来分辨实例对象的不同

in

in 可以通过对象上是否存在某个属性来分辨不同的对象

字面量类型保护

字面量类型保护跟 in 很相似是通过某个属性来分别不同的对象
